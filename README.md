# java-ui-testing
# Тестовое задание для практикума по автоматизации тестирования (SDET) от SimbirSoft.


*Исхаков Тимур*

## Описание


1. На языке программирования Java создан проект UI-автотестов по
тест-кейсам
2. В проекте использованы:
  - Selenium Webdriver (браузер Chrome)
  - При поиске элементов на странице селекторы: css, xpath, id
  - Тестовый фреймворк JUnit 4
  - Паттерн проектирования Page Object Model
  - Фреймворк для сборки проектов Maven
  - Формирование отчетов о пройденных тестах реализовано через Allure

__Описание тест-кейсов представлено в файле TestCases.pdf__
